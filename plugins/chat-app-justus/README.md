# Chat-App-JustUs Plugin

Starter plugin scaffold for the Chat-App-JustUs project.

## Installation

```bash
copilot plugin install chat-app-justus@awesome-copilot
```

## What's Included

- Plugin manifest: `.github/plugin/plugin.json`
- Starter docs for installation and contribution

## Next Steps

1. Add project artifacts (agents, instructions, skills) and reference them in `plugin.json`.
2. Run `npm run plugin:validate` to validate the plugin manifest.
3. Run `npm run build` to regenerate repository docs and marketplace metadata.
